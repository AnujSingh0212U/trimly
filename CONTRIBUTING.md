# Contributing

Thank you for your interest in Trimly!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/trimly.git`
3. Install dependencies: `npm install`
4. Copy `.env.example` to `.env` and configure
5. Run `npm run db:push && npm run dev`

## Pull Request Guidelines

- Use conventional commit messages (`feat:`, `fix:`, `docs:`, etc.)
- Ensure `npm run build` and `npm run test` pass
- Keep PRs focused on a single concern

## Code Style

- TypeScript strict mode, no `any` types
- ESLint + Prettier (run `npm run lint`)
- Follow existing patterns in services/repositories
